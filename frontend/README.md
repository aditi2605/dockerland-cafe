# ☕ DockerLand Café

> *Where Docker clicks like a perfectly pulled espresso shot.*

**Work in progress** — the kitchen is still being set up! 🍳

DockerLand Café is a fun, beginner-friendly website that teaches Docker through a café theme.
Confused by containers? Think of them as **cooked meals**. Dockerfile? That's your **recipe**.
Docker Hub? The **supermarket**. Suddenly Docker makes a lot more sense, right?

Currently brewing:
- 🧾 Bite-sized Docker lessons with café analogies
- 🍳 Hands-on labs with copy-paste commands
- 🧠 Quizzes to test what you've learned
- 📋 A Docker cheat sheet for quick reference

**Stack:** React · FastAPI · PostgreSQL · Docker (obviously 🐳)
